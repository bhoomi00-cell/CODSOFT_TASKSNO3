# CodSoft Python Internship - Task 3: Password Generator

## Description
A simple command-line based password generator built using Python. 
The program generates a strong, random password based on the length 
specified by the user, using a mix of uppercase and lowercase letters, 
digits, and special characters.

## Features
- User can specify desired password length
- Generates random passwords using letters (A-Z, a-z), digits (0-9), 
  and special characters
- Uses Python's `random` and `string` modules for secure randomness

## Technologies Used
- Python 3

## How to Run
1. Clone this repository
2. Run the file using: `python codsoftTask3.py`
3. Enter the desired password length when prompted
4. The generated password will be displayed on the screen
